# report-generator
Генератор Докладов

Военное RP-приложение для автоматического создания и управления докладами на постах, патрулях и специальных объектах.

О проекте

Генератор Докладов создан для упрощения работы сотрудников государственных структур на RP-проектах. Программа позволяет быстро формировать готовые доклады, отслеживать время нахождения на посту и получать напоминания о необходимости подачи очередного отчёта.

Основная цель проекта — сократить время на ручной ввод докладов и исключить ошибки при их составлении.
