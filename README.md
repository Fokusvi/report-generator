# Генератор Докладов

Военное RP-приложение для автоматического создания и управления докладами на постах, патрулях и специальных объектах.

---

## О проекте

Генератор Докладов создан для упрощения работы сотрудников государственных структур на RP-проектах.

Программа позволяет быстро формировать готовые доклады, отслеживать время нахождения на посту и получать напоминания о необходимости подачи очередного отчёта.

Основная цель проекта — сократить время на ручной ввод докладов и исключить ошибки при их составлении.

---

## Возможности

### Генерация докладов

- Автоматическое создание полного комплекта докладов.
- Доклады на заступление.
- Доклады на 20, 40 и 60 минут.
- Доклады на сдачу поста.
- Копирование текста одним нажатием.

### Поддерживаемые посты

- Штаб
- КПП-1
- КПП-2
- Дежурный по КПП
- Дежурный по части
- Выездной пост
- Совместный патруль с ССО
- Пеший патруль

### Пеший патруль

- Указание фамилии напарника.
- Автоматическая подстановка фамилии в текст доклада.
- Отображение дополнительного поля только при выборе пешего патруля.

### Выездной пост

- Выбор объекта.
- Автоматическое добавление объекта в текст доклада.

---

## Таймер и напоминания

- Автоматический отсчёт времени нахождения на посту.
- Напоминание о необходимости подачи очередного доклада.
- Звуковое оповещение.
- Возможность отключения звука одним нажатием.

---

## Сохранение данных

- Сохранение настроек между запусками программы.
- Восстановление активного поста после перезапуска.
- Сохранение выбранного состава.
- Сохранение выбранного кода.
- Сохранение состояния активных докладов.
